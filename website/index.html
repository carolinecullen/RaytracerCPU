Final Project

Created depth of field as well as animation with depth of field. I did the depth of field using a circular aperture and by taking 5 random samples on that aperture per pixel on the image and then also gave the image a focal length of where the aperture would focus to. I then used that same technigue was with super sampling to average the 5 color samples on the aperture. The python script was done using the subprocess library and then the gif was stitched together using ffmpeg.

Running Parameters for Final Project

  • "-focal" to run the depth of field program (required)
  • "-aperture=" to set the aperture size (required)
  • "-focalLength=" to set the focal length (required)
  • "-camera=0.0,0.0,0.0" to change the camera position (optional)
  • "-output=" to change the output file name (optional and defaults to output.png)
